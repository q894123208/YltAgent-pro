# 多科室冲突判断规则

当一个症状可能落入多个科室时，按以下规则给出 `primary_department` 与 `alternative_departments`。

## 1. 胸闷 / 胸痛
- 优先排查急诊：伴大汗、濒死感、放射至左肩背或下颌、持续 ≥ 15 分钟 → 急诊科。
- 伴心慌、活动后加重、既往有高血压/糖尿病/血脂异常 → 心血管内科。
- 伴咳嗽、咳痰、发热、活动后气短 → 呼吸内科。
- 不能区分且症状较轻 → 心血管内科 优先，呼吸内科 作为备选。

## 2. 头晕
- 伴一侧肢体无力、说话不清、意识改变 → 急诊科。
- 天旋地转、与头位变化相关、伴耳鸣听力下降 → 耳鼻喉科。
- 伴头痛、麻木、视物异常 → 神经内科。
- 伴心慌、胸闷、血压异常 → 心血管内科。
- 长期非特异性头晕、查不出明确原因 → 普通内科 / 全科医学科。

## 3. 腹痛
- 剧烈腹痛、板状腹、持续呕吐、便血 → 急诊科。
- 慢性、与进食或排便相关、伴腹泻或反酸 → 消化内科。
- 右下腹固定压痛、转移性右下腹痛 → 普外科。
- 育龄女性下腹痛、伴月经异常或白带异常 → 妇科。
- 孕期任何明显腹痛 → 产科（必要时急诊）。
- 儿童腹痛 → 儿科优先。

## 4. 腰痛
- 与活动姿势相关、有外伤史、放射至下肢 → 骨科。
- 一侧腰痛伴血尿、尿频尿急或绞痛 → 泌尿外科。
- 双侧腰部酸胀、伴水肿、泡沫尿、尿量异常 → 肾内科。
- 腰痛伴发热、寒战 → 急诊科或泌尿外科。

## 5. 小便疼
- 男性单侧腰痛 + 血尿 + 尿频 → 泌尿外科。
- 女性单纯尿频尿急尿痛、无白带异常 → 泌尿外科。
- 女性伴白带异常、外阴不适 → 妇科。
- 伴水肿、泡沫尿、血压高 → 肾内科。
- 儿童 → 儿科。

## 6. 咽痛 / 咳嗽
- 发热（≥ 37.3℃）伴咳嗽、咽痛、有流行病学史 → 发热门诊。
- 以咳嗽、咳痰、喘息为主，时间较长 → 呼吸内科。
- 以咽痛、声音嘶哑、扁桃体肿大、鼻塞、耳闷为主 → 耳鼻喉科。
- 儿童 → 儿科（高热抽搐立即急诊）。

## 7. 皮疹 / 瘙痒
- 全身荨麻疹伴面唇肿、呼吸困难、血压下降 → 急诊科。
- 局部皮疹、瘙痒、起红斑、脱屑 → 皮肤科。
- 皮疹伴关节肿痛、长期反复 → 风湿免疫科。
- 皮疹伴高热、精神差、瘀点瘀斑 → 急诊科 / 儿科。

## 8. 儿童症状（< 14 岁）
- 任何明显症状 → 儿科优先。
- 高热抽搐、呼吸急促、精神差、脱水 → 急诊科。
- 仅在儿科未开设的细分领域（如儿童眼科、儿童口腔），可由对应成人专科承接。

## 9. 孕期症状
- 已知怀孕或可能怀孕的女性 → 产科优先。
- 出血、剧烈腹痛、胎动减少 → 急诊科 / 产科急诊。
- 与怀孕无关的明确独立问题（如皮肤、口腔），可考虑相应专科，并提示告知孕期。

## 10. 症状不明确兜底
- 用户描述笼统（"浑身不舒服""没精神""乏力"）且无明确指向 → 普通内科 / 全科医学科。
- 多系统轻度症状 → 普通内科 / 全科医学科，由其分流。
- 体检异常但无明显症状 → 对应专科或普通内科。

## 11. 急诊优先原则
- 任何命中 `red_flags.md` 的情况 → `primary_department` = 急诊科。
- 其它疑似科室放入 `alternative_departments` 供后续随访。
